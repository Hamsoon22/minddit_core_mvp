# Minddit Core

Minddit Core는 워크숍, 교육 프로그램, 그룹 세션 등의 심리·정신건강 프로그램 운영을 위한 통합 관리 플랫폼입니다.프로그램 생성부터 참여자 관리, 실시간 세션 운영, 콘텐츠 관리, 리포트 확인까지 하나의 시스템에서 수행할 수 있도록 설계되었습니다.

---

## 주요 기능

### 대시보드

* 프로그램 현황 확인
* 참여자 통계 확인
* 일정 및 세션 관리

### 프로그램 관리

* 프로그램 생성 및 수정
* 세션 구성
* 프로그램 상태 관리

### 참여자 관리

* 참여자 등록
* 참여 현황 확인
* 참여자 정보 관리

### 콘텐츠 라이브러리

* 프로그램 자료 관리
* 활동 콘텐츠 저장 및 재사용

### 세션 운영

* 세션 진행 화면
* 활동 제어
* 참여자 인터랙션

---

## 기술 스택

* Next.js 14
* TypeScript
* Tailwind CSS
* Prisma
* NextAuth
* PostgreSQL

---

## 실행 방법

의존성 설치

```bash
npm install
```

개발 서버 실행

```bash
npm run dev
```

브라우저에서 아래 주소 접속

```text
http://localhost:3000
```
