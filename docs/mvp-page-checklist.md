# Minddit Core MVP 페이지 체크리스트

## 상태 정의
- 완료: 명세 기준 핵심 동작 구현 완료
- 진행중: 일부 UI/기능 구현됨
- 미착수: 주요 요구사항 미구현

## 1) 홈 (/dashboard)
- 상태: 진행중
- 완료
  - 요약 카드, 최근 프로그램/최근 일정 UI
  - 최근 일정 7일 선택 UI
- 미완료
  - 실제 ProgramSchedule 데이터 연동
  - 일정 선택 시 서버 데이터 기준 조회 정교화

## 2) 프로그램 관리 목록 (/sessions)
- 상태: 진행중
- 완료
  - 목록 조회 UI, 상태 표시
  - 새 프로그램 버튼/알림 UI
- 미완료
  - 프로그램 생성 스펙 입력(운영방식/일정유형) 연결

## 3) 프로그램 상세 (/sessions/[id])
- 상태: 진행중
- 완료
  - 상세 기본 정보 조회
- 미완료
  - 전체 일정/회차별 구성/기관정보 통합 표시
  - 메시지 생성 기능 연결

## 4) 프로그램 빌더 (/sessions/[id]/builder)
- 상태: 진행중
- 완료
  - 기본 빌더 UI 골격 존재
- 미완료
  - 일정유형별 일정 생성
  - 일정별 콘텐츠 추가/삭제/순서 변경/복제

## 5) 참여자 관리 목록 (/participants)
- 상태: 진행중
- 완료
  - 참여자 목록 UI, 새 참여자/알림 버튼
- 미완료
  - 참여자 생성/수정 API 연동 고도화

## 6) 참여자 상세 (/participants/[id])
- 상태: 진행중
- 완료
  - 상세 페이지 기본 조회
- 미완료
  - 프로그램 맥락 정보 및 안내 메시지 연계

## 7) 참여자 링크 페이지 (/s/[code])
- 상태: 진행중
- 완료
  - 링크 진입 경로 존재
- 미완료
  - 프로그램 소개/전체 일정/회차별 내용/오시는 길/문의처 완성

## 8) 콘텐츠 관리 (/library)
- 상태: 진행중
- 완료
  - 라이브러리 목록 UI
- 미완료
  - 콘텐츠 편집/검색/필터 강화
  - 빌더와의 삽입 워크플로 정식 연동

## 9) 설정 (/settings)
- 상태: 진행중
- 완료
  - 기관 정보 입력 UI, 토글 UI
- 미완료
  - 설정값 DB 저장/불러오기
  - 프로그램 생성 시 자동 연동 검증

## 10) 안내 메시지 생성 (신규 영역)
- 상태: 미착수
- 필요한 구현
  - 프로그램 전체 안내 메시지 템플릿
  - 회차별 안내 메시지 템플릿
  - 편집/복사 UI

## 우선순위
1. 프로그램 생성/수정 데이터 구조 확장
2. 일정유형 + 회차 구성 + 콘텐츠 배치
3. 참여자 링크 페이지 완성
4. 템플릿 메시지 생성
