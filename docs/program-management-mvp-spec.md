# Minddit Core - 프로그램 관리 기능 명세 (MVP)

## 목표

정신건강 전문가가 프로그램을 설계하고 운영하며, 참여자에게 프로그램 정보를 안내할 수 있도록 지원하는 프로그램 관리 기능을 구현한다.

---

# 1. 프로그램 관리

## 1-1. 프로그램 생성

사용자는 새로운 프로그램을 생성할 수 있다.

### 입력 항목

- 프로그램명
- 프로그램 설명 (선택)
- 운영 방식
  - 대면
  - 비대면
  - 혼합
- 운영 일정 유형
  - 주차별
  - 특정 일자별
  - 월별

---

## 1-2. 기관 정보 자동 연동

설정 페이지에 저장된 기관 정보를 자동으로 불러온다.

### 불러올 정보

- 기관명
- 주소
- 오시는 길
- 문의 전화
- 문의 이메일

프로그램별 수정 가능

---

# 2. 일정 구성

운영 일정 유형 선택 후 세부 일정을 생성할 수 있다.

### 예시

주차별

- 1주차
- 2주차
- 3주차
- 4주차

특정 일자별

- 2026.06.10
- 2026.06.17
- 2026.06.24

월별

- 6월
- 7월
- 8월

---

# 3. 프로그램 구성

각 일정(주차, 일자, 월)에 콘텐츠를 배치할 수 있다.

### 콘텐츠 출처

콘텐츠 관리 페이지의 라이브러리에서 선택

### 예시

1주차

- 감정 체크인
- 호흡 명상
- 감사 저널

2주차

- 바디스캔 명상
- 그룹 토론

### 제공 기능

- 콘텐츠 추가
- 삭제
- 순서 변경
- 복제

---

# 4. 참여자용 링크 생성

프로그램 생성 완료 시

참여자 전용 링크를 자동 생성한다.

### 기능

- 링크 복사
- 링크 공유
- QR 코드 생성 (향후 확장)

### 예시

minddit.com/program/abc123

---

# 5. 프로그램 수정

생성 이후에도 수정 가능

### 수정 가능 항목

- 프로그램명
- 프로그램 설명
- 운영 방식
- 일정
- 콘텐츠 구성
- 기관 정보

---

# 6. 프로그램 상세 페이지

프로그램 목록에서 선택 시 상세 페이지로 이동

### 표시 정보

- 프로그램명
- 설명
- 운영 방식
- 전체 일정
- 회차별 구성
- 참여자 수
- 생성일
- 상태
  - 초안
  - 예정
  - 진행중
  - 완료

---

# 7. 안내 메시지 자동 생성

AI 생성이 아닌 템플릿 기반 자동 생성 방식으로 구현

프로그램 정보와 기관 정보를 조합하여 자동 생성한다.

### 제공 기능

- 메시지 생성
- 편집
- 복사

---

## 7-1. 프로그램 전체 안내 메시지

버튼

[안내 메시지 생성]

### 자동 삽입 정보

- 프로그램명
- 전체 일정
- 프로그램 링크
- 기관 정보

### 생성 예시

안녕하세요.

'{{프로그램명}}' 참여를 신청해주셔서 감사합니다.

본 프로그램은 {{시작일}}부터 {{종료일}}까지 진행됩니다.

회차별 세부 일정 및 활동 내용은 아래 링크에서 확인하실 수 있습니다.

{{프로그램링크}}

문의사항은 아래 연락처로 연락 부탁드립니다.

문의전화 : {{전화번호}}
이메일 : {{이메일}}

감사합니다.

# 8. 참여자 링크 페이지 (중요)

참여자가 링크를 통해 접속하는 페이지

### 목적

운영자가 반복적으로 안내하지 않아도 필요한 정보를 확인할 수 있도록 함

### 제공 정보

- 프로그램 소개
- 전체 일정
- 회차별 내용
- 활동 안내
- 오시는 길
- 문의처
- 진행 방식

### 기대 효과

- 문의 감소
- 안내 메시지 단순화
- 운영 효율 증가

운영자는 링크만 전달해도 대부분의 정보를 안내할 수 있다.
